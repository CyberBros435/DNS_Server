# DNS LOOKUP UTILITY - PROJECT DOCUMENTATION
============================================================

A professional, lightweight Python-based Command Line Interface (CLI) 
tool designed to perform DNS lookups using the system's native 
'nslookup' capabilities. This tool features a stylized ASCII 
interface and color-coded terminal output for an enhanced user experience.

------------------------------------------------------------
## 🚀 FEATURES
------------------------------------------------------------
* Interactive CLI: User-friendly prompts for continuous lookups.
* Visual Feedback: Uses 'colorama' for colored status messages.
* Automated Lookup: Quickly retrieves IP addresses and domain records.
* Custom Banner: Stylized ASCII art greeting upon launch.
